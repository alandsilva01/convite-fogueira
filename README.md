# Fogueira com Jesus — Pre Teams

Convite interativo otimizado para celular.

## Como publicar no GitHub Pages (recomendado, gratuito)
1. Crie uma conta em https://github.com (se ainda não tiver).
2. Clique em **New repository**, dê um nome (ex: `fogueira-com-jesus`) e marque como **Public**. Não marque "Add README".
3. Na página do repositório vazio, clique em **uploading an existing file**.
4. Arraste todos os arquivos desta pasta (`index.html`, `style.css`, `script.js` e a pasta `assets` inteira) para a área de upload.
5. Clique em **Commit changes**.
6. Vá em **Settings > Pages** (menu lateral esquerdo).
7. Em "Branch", selecione `main` e pasta `/ (root)`, clique em **Save**.
8. Aguarde 1–2 minutos. O link ficará assim:
   `https://SEU-USUARIO.github.io/fogueira-com-jesus/`
9. Esse é o link simples para enviar aos pais.

## Alternativa ainda mais rápida: Netlify Drop (sem precisar de conta)
1. Acesse https://app.netlify.com/drop
2. Arraste a pasta `fogueira-com-jesus-netlify-final` (ou o ZIP) para a página.
3. Aguarde gerar o link (algo como `nome-aleatorio.netlify.app`).
4. Envie o link no WhatsApp.

## Já configurado
- Botão de localização: Rua Campinas do Sul, 141
- Botão de confirmação: via formulário do Google (ícone e texto atualizados, não é mais WhatsApp)
- Botão de agenda: 18/07 às 18h
- Música em loop, com tentativa de autoplay assim que a página carrega (toca mudo e desmuta na primeira interação, caso o navegador bloqueie o som direto)
- Botão discreto de música no canto inferior direito
- Contagem regressiva no canto superior direito (não sobrepõe mais a logo)
- Botões de ação centralizados no poster
- Layout responsivo: encaixa inteiro na tela em qualquer celular, sem cortes nem scroll
- Animação leve de estrelas/brasas caindo

Observação: alguns navegadores (principalmente dentro de apps como WhatsApp/Instagram) bloqueiam áudio com som até qualquer toque na tela. Nesses casos a música começa mudo e desmuta automaticamente assim que a pessoa tocar em qualquer lugar da tela — o que normalmente já acontece ao abrir o link.
